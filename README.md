# Quadratic Function Grapher

A simple Python desktop app that draws the graph of a quadratic function **f(x) = ax² + bx + c** on a coordinate plane using tkinter.

## Features

- Draws x and y axes with labeled tick marks
- Plots any quadratic function based on user-defined parameters
- Customizable canvas size
- Clean red curve rendered on a coordinate grid

## Requirements

- Python 3.x
- tkinter (included with standard Python installation)

## How to Run

```bash
python graf.py
```

When prompted, enter:
1. Canvas width (e.g. `600`)
2. Canvas height (e.g. `600`)
3. Parameters **a**, **b**, **c** for the function f(x) = ax² + bx + c

## Example

```
Canvas width: 600
Canvas height: 600
a: 1
b: 0
c: 0
```

This will draw the parabola **f(x) = x²**.
